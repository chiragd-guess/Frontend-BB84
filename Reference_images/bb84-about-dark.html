About BB84 // Sim

Quantum secure communication using the BB84 protocol

BB84 Sim is a quantum secure messenger simulator that demonstrates the BB84 Quantum Key Distribution protocol. It lets two parties generate a shared secret key using the principles of quantum mechanics, then exchange encrypted messages with unconditional security.
